## 📖 نبذة عن المشروع

منصة تعليمية تفاعلية متكاملة لتعليم الأطفال **الحروف الإنجليزية** بطريقة **الصوتيات البحتة (Pure Phonetics)**.  
تم تصميم المنصة خصيصاً للأطفال لتكون **صديقة للعين (eye-friendly)** و **تفاعلية (interactive)**، مع استخدام **صور وأصوات حقيقية من الإنترنت**.

> 🎯 **الهدف**: تعليم الأطفال القراءة والكتابة من خلال دمج المقاطع الصوتية (blending) بطريقة تصاعدية ومنطقية.

## ✨ المميزات الرئيسية

| الميزة | الوصف |
|--------|-------|
| 🔤 **26 حرفاً** | جميع حروف الأبجدية الإنجليزية من A إلى Z |
| 🎮 **40+ لعبة** | أكثر من 40 لعبة تفاعلية متنوعة لكل حرف |
| ✍️ **تتبع الحروف** | تعلم كتابة الحروف بالتتبع بالإصبع أو الفأرة |
| 🖼️ **صور من الإنترنت** | معرض كلمات مع صور حقيقية من Pixabay (CDN مجاني) |
| 🔊 **نطق صوتي** | نطق الحروف والكلمات باستخدام Web Speech API |
| 📱 **تصميم متجاوب** | يعمل على جميع الأجهزة (جوال، جهاز لوحي، كمبيوتر) |
| 🎨 **واجهة صديقة للأطفال** | ألوان مريحة، أزرار كبيرة، تصميم جذاب وآمن |
| 🌐 **بدون خادم** | يعمل محلياً أو عبر GitHub Pages |
